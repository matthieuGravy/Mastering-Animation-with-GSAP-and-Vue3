# Mastering Animation with GSAP and Vue 3
## open source

This project explores the integration of the powerful animation library GSAP with Vue 3. Discover how to create smooth and interactive animations in your Vue 3 applications using GSAP. Explore concrete examples and best practices to breathe life into your user interfaces with dynamic transitions and visual effects.

If you want to learn more about GSAP, visit <a href="https://greensock.com/gsap/">here.</a>

### Project Setup

```sh
npm install
```

<h4>Compile and Hot-Reload for Development</h4> 

```sh
npm run dev
```

<h4>Compile and Minify for Production</h4> 

```sh
npm run build
```

<h4>Lint with [ESLint](https://eslint.org/)</h4> 

```sh
npm run lint
```
